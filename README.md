# Employee Turnover Risk Modeling

This repository contains the analytical pipeline developed for the master's thesis:
"Analysis and Modeling of Employee Turnover Risks in a Graded Compensation System".
The project explores how characteristics of the compensation system and employee attributes influence the probability of voluntary turnover.

## Research Objective
The objective of this research is to analyze factors associated with employee turnover and to develop predictive models for estimating the risk of employee attrition.
Special attention is given to compensation-related variables within a graded salary structure.

## Methodology
The study applies a combination of statistical and machine learning approaches, including:
- exploratory data analysis (EDA)
- feature engineering for HR data
- survival analysis
- machine learning models for turnover prediction
- model evaluation and comparison

## Data Availability

The dataset used in this research contains confidential HR information and cannot be publicly shared due to NDA restrictions.
Therefore this repository contains only the analytical code and documentation required to reproduce the methodology.

## Repository Structure

notebooks/ – exploratory analysis and modeling experiments

src/ – reusable scripts for data processing and modeling

docs/ – research documentation

results/ – figures and model evaluation outputs
